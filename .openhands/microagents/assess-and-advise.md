# Assess and Advise Microagent

This microagent provides assessment and advisory capabilities for the AutomatedTrendTrackerContentCreator project.

## Purpose

The assess-and-advise microagent helps users by:

1. **Assessing** the current state of code, configurations, or project components
2. **Advising** on best practices, improvements, and recommendations

## Capabilities

### Code Assessment
- Review code quality and structure
- Identify potential issues or areas for improvement
- Evaluate adherence to coding standards and best practices

### Project Assessment
- Analyze project architecture and organization
- Review dependencies and their versions
- Assess documentation completeness

### Advisory Services
- Provide recommendations for code improvements
- Suggest best practices for the project domain
- Offer guidance on implementation approaches
- Recommend solutions for identified issues

## Usage

When activated, this microagent will:

1. Analyze the relevant code, configuration, or project component
2. Identify strengths and areas for improvement
3. Provide clear, actionable recommendations
4. Explain the reasoning behind each suggestion

## Guidelines

- Assessments should be thorough but focused
- Advice should be practical and actionable
- Recommendations should consider the project context
- Suggestions should align with industry best practices
